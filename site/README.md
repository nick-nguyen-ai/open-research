# site/ — OpenResearch static site (M2)

Astro 5 + Tailwind 4 + React islands + Pagefind. Reads `../content/` — never edit
site data by hand.

- `npm run site:dev` (repo root) — derive + dev server on :4321. Search needs a
  full build; the overlay says so in dev.
- `npm run site:build` — derive → astro build → pagefind. Output in `site/dist/`.
- `node site/scripts/check-links.mjs site/dist` — route/link/index checks (CI-blocking).
- Tests: `npm test` (workspace runs bare `node --test` — do not pass a directory).

Design authority: `docs/superpowers/specs/2026-07-13-m2-site-design.md`.
Tokens live in `src/styles/global.css`; re-skin = edit `:root` custom properties.
`src/data/*.json` is generated by `scripts/derive.mjs` and git-ignored.

## Motion QA checklist

Manual check after any change touching motion, theme, or search — serve `dist/`
(e.g. `npx serve site/dist`) and verify:

- [ ] Home stats tick up once; cards stagger in.
- [ ] Card → article click morphs the title (view transition, Chromium).
- [ ] Rail annotations settle in on scroll; stamps press in last; scroll-spy highlights TOC.
- [ ] ⌘K overlay scales in; result match underlined oxblood; Esc closes.
- [ ] Theme toggle flips instantly, persists across navigation and reload.
- [ ] With OS reduced-motion ON: no animation anywhere, all content visible.
- [ ] With JS disabled: all pages fully readable (no invisible `.rv` content).
